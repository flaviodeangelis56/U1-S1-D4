Come creare un tamplate per i post


di mario rossi - pubblicato il 25-10-2020


La prima cosa da fare é suddividere in blocchi logici la pagina, identificando le parti comuni da applicare a tutte le pagine


Le varie aree sranno suddivise in:


  • contenuti di intestazione del tag <header>
  • contenuti centrali variabili del tag <main> (corpo principale)
  • contenuti di fine pagina comuni,da inserire nel tag<footer>

Formattazione del codice il testo dell'articolo dovra essere formattato per identificare tutte le parti da selezionare con CSS. Perciò ipassi saranno:

  1. Progettare il markup assegnando ad ogni specifico testo il tag appropriato
  2. Scrivere l'HTML dai tag esterni a quelli interni e dall'alto vero il basso
  3. Applicare gli stili CSS

L'applicazione di stili CSS é cresciuta negli anni fino a raggiungere la diffusione pressochè totlae
